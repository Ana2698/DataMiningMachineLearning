In this project machine learning model for the Titanic dataset is created.
The dataset can be found on kaggle using the link:-
https://www.kaggle.com/competitions/titanic/data
*******Steps to follow to run x2020dxz.ipynb file***********
OS: Windows 10
IDE: Jupyter notebook

Follow the instructions below to run .ipynb file: -
1. Download and Install Anaconda Navigator.
2. Open jupytor notebook.
3. Upload x2020dxz.ipynb file to Jupyter.
4. download and load the train and test data from kaggle before running the code.
